# The mobile factory :iphone:

E-commerce platform dedicated to smartphones. In the app, customers can create a user account to start adding items to their shopping carts and finally make purchases through the checkout process üíµ. Additionally, there are admin/owner users who have privileges to edit categories, add/remove products, and verify orders for all customers.


## Installation :hammer:

git clone https://github.com/MatiasOrlando/the-mobile-factory.git

cd back | cd front
--------|---------
npm i   | npm i

cd back     | cd front
------------|---------
npm run server | npm run dev
